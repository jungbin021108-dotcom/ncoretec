# 업로드 가이드 (GitHub Pages)

1) GitHub 접속 → 로그인
2) 새 저장소 생성: 사용자명.github.io (예: jeongbin.github.io)
3) 이 폴더 안의 모든 파일과 images 폴더를 **그대로 업로드**
   - 방법: 저장소 → Code → Add file → Upload files → 드래그 앤 드롭 → Commit changes
4) Settings → Pages → Branch: main, Folder: /(root) → Save
5) 1~3분 뒤 https://사용자명.github.io 접속 → 사이트 확인

수정 방법
- 디자인 변경: style.css 편집 후 Commit changes
- 내용 변경: 각 .html 파일을 편집한 뒤 Commit changes
- 로고 교체: images/logo.svg 파일을 본인 로고로 교체(파일명 동일 유지 권장)

문서/영상 연결
- catalog.html / manual.html에 PDF 링크 추가
- videos.html에 유튜브 iframe 또는 mp4 링크 추가 (대용량은 유튜브 권장)
